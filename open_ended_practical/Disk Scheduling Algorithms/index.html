Disk Scheduling Algorithms

Visualize and calculate the Total Head Movement required to satisfy disk I/O requests.

FCFS

First Come First Serve

Services requests in strict order of arrival. Simple but high head movement.

SSTF

Shortest Seek Time First

Greedy approach. Picks the closest request to the current head position.

SCAN

Elevator Algorithm

Moves head from one end to the other, servicing all requests in between.

C-SCAN

Circular SCAN

Provides uniform waiting time by only servicing requests in one direction.
